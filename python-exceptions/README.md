# Python - Exceptions

This directory contains Python scripts demonstrating exception handling.

## Tasks

### Mandatory Tasks

0. **Safe list printing** - `0-safe_print_list.py`
   - Prints x elements of a list safely using try/except

1. **Safe printing of an integers list** - `1-safe_print_integer.py`
   - Prints an integer with "{:d}".format() safely

2. **Print and count integers** - `2-safe_print_list_integers.py`
   - Prints the first x elements of a list and only integers

3. **Integers division with debug** - `3-safe_print_division.py`
   - Divides 2 integers and prints the result

4. **Divide a list** - `4-list_division.py`
   - Divides element by element 2 lists

5. **Raise exception** - `5-raise_exception.py`
   - Raises a type exception

6. **Raise a message** - `6-raise_exception_msg.py`
   - Raises a name exception with a message

### Advanced Tasks

100. **Safe integer print with stderr** - `100-safe_print_integer_err.py`
    - Prints an integer with error handling to stderr

101. **Safe function** - `101-safe_function.py`
    - Executes a function safely

102. **Magic calculation** - `102-magic_calculation.py`
    - Does exactly the same as the Python bytecode

## Requirements

- All files must be executable
- All files must end with a new line
- Code should use try/except blocks where appropriate
- No module imports allowed (except where specified)
