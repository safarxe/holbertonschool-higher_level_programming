# Python - Classes and Objects

This directory contains Python scripts demonstrating object-oriented programming concepts with classes and objects.

## Tasks

### Mandatory Tasks

0. **My first square** - `0-square.py`
   - Defines an empty class Square

1. **Square with size** - `1-square.py`
   - Defines a class Square with a private size attribute

2. **Size validation** - `2-square.py`
   - Adds size validation to the Square class

3. **Area of a square** - `3-square.py`
   - Adds area calculation method to the Square class

4. **Access and update private attribute** - `4-square.py`
   - Adds property getter and setter for size attribute

5. **Printing a square** - `5-square.py`
   - Adds method to print the square with # character

6. **Coordinates of a square** - `6-square.py`
   - Adds position attribute and updates printing method

## Requirements

- All files must be executable
- All files must end with a new line
- Code should use pycodestyle (version 2.7.*)
- All modules and classes must have documentation
- No module imports allowed (except where specified)
